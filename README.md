# NAV Integrity backend coding challenge 

### Steps

1. Install Node.js and NPM on specific required versions:
 - node version 14.x.x
 - npm version 6.x
2. Install JavaScript dependencies
     ```sh
   npm install
   ``` 
3. Run server 
     ```sh
   npm start
   ``` 
4. Run index.js file 
    ```sh
   npm test
   ```
